# **App Name**: pamoja

## Core Features:

- Crisis Toolkit: Provide quick access to breathing exercises and safety plans.
- Emotion Diary: Track daily emotions with a mood dial and logging features.
- Thought Reframe Tool: AI-powered tool to provide validating reframes for negative thoughts, using ChatGPT.
- Penpal Connection: Enable connection between users for support and sharing.

## Style Guidelines:

- Primary color: Indigo (#4B0082) to evoke calm and gentleness.
- Background color: Purple (#800080) to create a non-clinical feel.
- Accent color: Navy Blue (#000080) for a refreshing touch.
- Body and headline font: 'Literata', a serif font for a literary and vintage feel.
- Use gentle and rounded icons for a soft interface.
- Incorporate subtle fade-in animations to transition between screens.
